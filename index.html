Funcionalidades

Feito para o fluxo do estoque

Cadastro completo

Registre titulo, autor e capa de cada livro.

Movimentacoes

Controle entradas e saidas individuais ou em lote, com motivo obrigatorio.

Estoque seguro

O sistema bloqueia saidas que deixariam algum livro com quantidade negativa.

Relatorios

Acompanhe historico de alteracoes com data, usuario, tipo e quantidade.

Busca e filtros

Encontre livros por titulo ou autor e destaque itens com estoque baixo.

Exportacao

Baixe uma planilha Excel com o estoque atual.

Arquitetura

GitHub Pages para apresentacao, Vercel para uso real

Esta pagina serve como vitrine do projeto. A aplicacao principal roda na Vercel, onde tambem existe a API serverless que conversa com o banco Upstash Redis.
